
# scrool-animation
---
##  Overview

This project is a **modern animated hero section** built using **React (Next.js) and GSAP**.
It demonstrates **smooth UI animations**, **scroll-based interactions**, and **performance optimization**.

The goal of this project is to create a **premium landing experience** where animations are controlled by user scroll instead of time-based triggers.

---

##  Features

### Hero Section Layout

* Full screen (above-the-fold) hero section
* Letter-spaced animated headline
* Impact metrics (Success %, Projects, Clients)

---

### Initial Load Animation

* Smooth fade + upward motion
* Letter-by-letter animation using **stagger effect**
* Stats appear sequentially

---

### Scroll-Based Animation (Core Feature)

* Animation controlled by **scroll position**
* Hero section **pinned during animation**
* Main object moves across screen smoothly
* Content fades and reappears
